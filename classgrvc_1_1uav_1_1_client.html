Detailed Description

Cient interface for other modules to communicate with a uav (or uav-derived) node.

Different implementations of this interface are responsible of communications with an analogous uav service. For example, uav::ClientRos (derived from uav::Client), implements communications using topics and services and expects a uav::SericeRos to be running in some other node in the network. Equivalent, uav::ClientHttp would require uav::ServiceHttp to be running somewhere, with an accessible IP address.
